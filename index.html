Simulation-based Inference at scale for Particle Physics

NEural-basEd Diffusion Likelihood Estimations (NEEDLE) is an inter-experiment research project, aimed at developing a computational efficient statistical inference toolkit, using neural-based diffusion and flow matching models, for analysing proton-proton collision data from the Large Hadron Collider (LHC). Our project aims to package neural likelihood, posterior, and likelihood ratio estimation using thermo-equilibirum inspired models (diffusion/flow matching) with scalable orchestration compute infrastructure, to unlock simulation-based inference in particle physics.

By leveraging DAG-based workflow and modern Machine Learning (ML) frameworks, we're building the next generation of tools that will help physicists explore fundamental questions about the universe.

Research Focus

Advanced ML algorithms for particle collision analysis, including deep learning models for event classification, anomaly detection, and feature extraction from high-dimensional detector data.

Infrastructure

DAG-based orchestration system designed for reproducible, scalable ML workflows. Our architecture supports distributed training, efficient data pipelines, and seamless integration with LHC computing resources.

Collaboration

Multi-institutional effort bringing together expertise in particle physics, machine learning, and high-performance computing.
