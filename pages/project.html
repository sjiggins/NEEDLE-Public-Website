Project Overview

Mission

The NEEDLE (Next-gEneration Event-Data Learning Environment) project aims to revolutionize how particle physicists analyze collision data from the Large Hadron Collider. By combining cutting-edge machine learning techniques with robust, scalable infrastructure, we're building tools that enable breakthrough discoveries in fundamental physics.

Our work focuses on three core pillars: developing advanced ML models tailored for particle physics, creating efficient orchestration systems for massive-scale data processing, and fostering collaboration between the physics and AI communities.

Architecture Design

DAG-Based Orchestration

Our system uses Directed Acyclic Graphs (DAGs) to represent complex ML workflows, enabling parallel execution, dependency management, and fault tolerance. Each node in the DAG represents a discrete computational task (data loading, preprocessing, training, evaluation), with edges defining data dependencies.

Key Features

  • Dynamic Scaling: Auto-scaling workers based on queue depth
  • Resource Management: Intelligent GPU/CPU allocation
  • Caching: Intermediate results cached for rapid iteration
  • Checkpointing: Automatic recovery from failures
  • Versioning: Full lineage tracking for reproducibility

Integration Points

  • CERN computing grid (WLCG) for distributed processing
  • Cloud resources (AWS, Google Cloud) for burst capacity
  • Local cluster support for development and testing

Machine Learning Research

Current Research Areas

1. Particle Identification

Developing transformer-based architectures for identifying particles in detector data. Our models leverage self-attention mechanisms to capture long-range correlations in calorimeter and tracker measurements, achieving 95%+ accuracy on challenging signatures.

2. Jet Reconstruction

Graph Neural Networks (GNNs) for reconstructing jets from particle flow candidates. Our approach treats detector hits as graph nodes, learning optimal clustering and energy assignment through message passing.

3. Anomaly Detection

Unsupervised learning for model-independent searches of new physics. Using variational autoencoders and normalizing flows to identify rare, anomalous collision events that may indicate beyond-Standard-Model phenomena.

4. Real-Time Inference

Optimizing models for deployment in the LHC trigger system. Achieving sub-millisecond latency through quantization, pruning, and hardware-aware neural architecture search on FPGA targets.
