Project Overview

Mission

The NEEDLE (NEural-basEd Diffusion Likelihood Estimations) projects aims to develop a new simulation-based inference toolkit, that utilises thermo-equilibrium inspired models, such as diffusion and flow matching, for the high energy particle physics community.

Our work focuses on three core pillars: developing new neural-based surrogates, developing new likelihood/posterior inference methodologies within the diffusion/flow-matching domain, and creating an efficient data ingestion and orchestration system for highly parallelisable HTC/HPC systems. The work herein is at present, the collaboration of particle physics groups within the ALICE, ATLAS, and CMS Collaborations at the European Center for Nuclear Research (CERN).

Architecture Design

DAG-Based Orchestration

Our system uses Directed Acyclic Graphs (DAGs) to represent complex ML workflows, enabling parallel execution, and dependency management. Each node in the DAG represents a discrete computational task (data loading, preprocessing, training, evaluation), with edges defining data dependencies.

Machine Learning Research

Current Research Areas

1. Diffusion/Flow-matching Models

Developing new diffusion/flow-matching models for likelihood and posterior inference tasks. Optimising inference speed using latest developments in numerical ODE/SDE solvers.

2. Neural Surrogates

Transformer-based neural surrogates, leveraging the attention mechanisms to learn conditionality, and aid in interpretability.

3. Real-Time Inference

Optimizing transformer model architectures to efficiently ingest irregular data sizes, in order to reduce memory footprint of ML training and inference tasks.
