Real-Time Inference

Sub-millisecond ML inference for LHC trigger systems and online event selection

Overview

Real-time inference systems enable ML models to run directly in the LHC trigger, making split-second decisions on which collision events to save for analysis. Operating at 40 MHz collision rates, these systems must achieve sub-microsecond latency while maintaining high accuracy.

Physics Impact

Trigger Applications

  • Jet Tagging: b-tagging, tau identification in real-time
  • Missing Energy: Fast neutrino reconstruction
  • Anomaly Detection: Model-independent BSM searches
  • Particle ID: Real-time electron/photon discrimination

Discovery Potential

ML-enhanced triggers increase sensitivity to rare processes by 30-50%, enabling discoveries that would be missed by traditional trigger algorithms. This is particularly crucial for long-lived particles and other non-standard signatures.
